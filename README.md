# IPL Data Analysis — Uncovering Patterns from 2008 to 2024

## Project Overview
Analyzed IPL match data from 2008 to 2024 using Python and Pandas 
to uncover interesting patterns and insights.

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Insights
- Mumbai Indians are the most successful team with 144 wins
- Chennai Super Kings are second with 138 wins
- Winning the toss does not guarantee winning the match — only 50.59%
- AB Russell is the top Player of the Match winner with 15 awards

## Dataset
- Source: Kaggle — IPL Complete Dataset
- matches.csv — Match level data
- deliveries.csv — Ball by ball data

## Dataset Source
Download from Kaggle: https://www.kaggle.com/datasets/patrickb1912/ipl-complete-dataset-20082024
